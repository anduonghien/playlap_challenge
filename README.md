# Playlab challenge

A slution read file log for Playlab challenge.

## Usage

```shell
$ git clone git@github.com:dhienan/playlap_challenge.git
$ cd playlap_challenge
```
### Run it

```shell
$ ruby source/play_lab_challenge.rb
```
### Run Rspec

```shell
$ bundle install
$ bundle exec rspec
```
